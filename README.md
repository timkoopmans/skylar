# Skylar

A load testing tool for ScyllaDB

    skylar -h

    Usage: skylar [OPTIONS]

    Options:
    --read-threads <READ_THREADS>    Number of read threads [default: 10]
    --write-threads <WRITE_THREADS>  Number of write threads [default: 90]
    -h, --help                       Print help
